# Streaming_Platfrom_SQL_Case_Study
📌 Project Overview

This project analyzes user behavior, content engagement, subscription patterns, and retention metrics for a streaming platform using SQL. The objective is to derive actionable insights that can help improve user engagement, customer retention, and revenue growth.

The analysis simulates a real-world OTT platform such as Netflix by leveraging relational data across users, subscriptions, content, and viewing history.
